Requirements
============
* PHP 5.3.7+
* MySQL 5 database
* activated mysqli extension

Installation
============
Create a database *login* and the table *users* via the SQL statements in the `_install` folder.  
Change mySQL database user and password in `config/db.php` (*DB_USER* and *DB_PASS*).
